\## 프로젝트 개요



Unity 6 기반 인터랙티브 3D 콘텐츠 UI 프로토타입입니다.  

전시·키오스크 환경을 가정하여 콘텐츠 선택, 상세 보기, 3D 모델 조작, 퀴즈, 결과 화면, 자동 복귀 흐름을 구현했습니다.



\- 개발 기간: 핵심 기능 구현 약 4일

\- 개발 형태: 개인 프로젝트

\- 구현 범위: UI 흐름 설계, 패널 전환, 3D 모델 인터랙션, JSON 퀴즈 데이터 연동, 자동 복귀 및 페이드 전환

\- 사용 기술: Unity 6, C#, UGUI, TextMeshPro, Input System, JSON
